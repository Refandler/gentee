pvmEngine  STDCALL gentee_init( uint flags );
Создает копию Gentee VM.
 Принемает:
    uint flags - флаги описания VM (можно совмещать)
                   G_CONSOLE - создает консольный вывод
                   G_SILENT  - создает без дисалейноый вид (вывод надо перехватывать)
                   G_CHARPRN - Вывод в Windows кодировки
 Возвращает:
    pvmEngine - указатель на созданную VM.



uint  STDCALL gentee_deinit( pvmEngine pThis );
Удаляет копию Gentee VM.
 Принемает:
    pvmEngine pThis - указатель на VM (удаляемую)
 Возвращает:
    uint - TRUE - успех, FALSE - неудача. 



uint  STDCALL gentee_load( pvmEngine pThis, pubyte bytecode, uint flag );
Функция загружает байт код в VM.
  Принемает:
     pvmEngine pThis - указатель на VM 
     pubyte bytecode - Указатель на данные, или имя файла с данными байт кода (зависит от флагов)
     uint flag       - Флаги описания загружаемых данных
                         GLOAD_ARGS  - Пердать текущие аргументы коммандной строки в VM.
                         GLOAD_FILE  - указывает, что указатель на данные это имя файла - откуда надо грузить данные иначе сами данные.
                         GLOAD_RUN   - Флаг указывает, что после загрузки нужно ваполнить main функцию.
  Возвращает:
     uint - Результат вернувшее функция main (если -1 возможно была ошибка)



uint STDCALL gentee_set( pvmEngine pThis, uint state, pvoid val );
Изменяет внутреннии пременные VM
  Принемает:
     pvmEngine pThis - указатель на VM 
     uint state      - Флаг, что менять (нельзя обьединять)
                         GSET_TEMPDIR  - Specify the custom temporary directory
                         GSET_PRINT    - Specify the custom print function
                         GSET_MESSAGE  - Specify the custom message function
                         GSET_EXPORT   - Specify the custom export function
                         GSET_ARGS     - Specify the command-line arguments
                         GSET_FLAG     - Specify flags 
                         GSET_DEBUG    - Specify the custom debug function
                         GSET_GETCH    - Specify the custom getch function
     pvoid val       - Указатель на данные который надо установить (строки не копируются)
                       при GSET_FLAG передаются сами флаги, а не указатель на них.
  Возвращает:
     uint - TRUE - успех, FALSE - неудача.
