Некотарая справка
 При закрытии дебагера выполняется запуск автоматически (если сработал брейк)
 (Ctrl - пробел) Хоткей автодополнения или подсказки если слево от курсора открывающая скобка
 F3 - продолжить поиск
 Первая вкладка меняется только из Open в тулбаре (её закрыть нельзя)

При активном отладчике доступны ф-ии
  dbgPrint(str) - печать лога в консоль дебагера
  dbgBreak()    - брейк на следущей строке


============================ Использование Dll =================================================================
 Запускается отладчик через geDebuger_Init(setupDebuger *st);
		передается структура
	typedef struct _setupDebuger
	{
	    HWND            mainWND;
	    unsigned long   flag; 
	    _gentee_init    ge_init;
	    _gentee_compile ge_compile;
	    _gentee_set     ge_set;
	    _gentee_ptr     ge_ptr;
	    _gentee_call    ge_call;
	}setupDebuger;
	где 
	mainWND - хендл окна родителя (NULL - если нет)
	flag - флаги управления
		DEBUG_CHILD_WINDOWS - отладчик встраивается как контрол (если задано mainWND)
		DEBUG_HIGE_FILELIST - Скрыть список файлов (проводник)
		DEBUG_HIGE_DEBUGLOG - Скрыть консоль отладчика
		DEBUG_SHOW_WINDOWS - Создать сразу видемое окно отладчика (не реализовано)
	ge_init, ge_compile, ge_set, ge_ptr, ge_call - указатели на соотведствующие функции gentee (нужно для перехвата)

geDebuger_Destroy() - остановить отладчик (окно убивается)
geDebager_Show(BOOL isView) - скрыть\покозать окно отладчика
geDebuger_Move(int x, int y, int w, int h) - изменения позиции и размера окна отладчика
	x, y - координата верхней левой точки окна отладчика
        w,  h - ширина, высота окна

Примеры использования в exe.c и TestDll.rar

Некоторые аспекты реализации
 - окно отладчика (всё ядро) работает в другом треде (он убивается только при вызове geDebuger_Destroy())
 - онализ скрипта идет только по активному окну редактора (включая инклуды).
 - отладчик убирает влаг CMPL_THREAD.
 - Установка всех калбечных функций работает (только отладчик получает управление первым)
 - Стандартный вывод компилятора переводится на консоль дебагера
 - Отладчик совместим только с версиями gentee 3.0 до 3.2 (Возможно другие кроме 3.3 (недоступны дебажные функции))

В дебагере пока не доделано:
 - поиск по файлам
 - трейс стека VM
 - Выполнить до курсора
 - старый анализ кода (от 3.1 думаю проабдейтит на следущей версии компилятора)
